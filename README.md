Lista 2 - Exercícios de JavaScript

Nesse repositório esta a Lista 2 de exercícios feitos em JavaScript, utilizando estruturas de repetição (for), condições (if) e cálculos com variáveis.

Exercícios

1. Sorteio de números
Sorteia 20 números entre 0 e 10 e:
- Mostra todos os números sorteados
- Conta quantos são maiores que 5
- Conta quantos são divisíveis por 3

2. Idades de pessoas
Lê a idade de 10 pessoas e:
- Calcula a média das idades
- Conta quantas são maiores de 18 anos
- Conta quantas são menores de 5 anos
- Mostra a maior idade informada

3. Idade e sexo
Lê idade e sexo de 5 pessoas e mostra:
- Quantidade de homens e mulheres
- Média de idade do grupo
- Média de idade dos homens
- Quantas mulheres têm mais de 20 anos

4. Peso e altura
Lê peso e altura de 7 pessoas e:
- Calcula a média de altura
- Conta pessoas com mais de 90 kg
- Conta pessoas com menos de 50 kg e altura menor que 1,60 m
- Conta pessoas com mais de 1,90 m e mais de 100 kg

Linguagem utilizada

- JavaScript
